# Retenciones impositivas — Boreal

## Alcance

Este anexo complementa las reglas de aprobación ya vigentes para el cliente y describe qué hacer cuando el agente de recaudación informa un comprobante que no coincide con el importe acreditado.

## Criterio de tolerancia

El responsable abre el legajo del período, verifica que la carga esté completa y deja asentada la revisión con su usuario y la fecha. Si falta documentación, la solicita antes de continuar y el asiento queda en espera. El resultado se registra en la misma línea, de modo que una revisión posterior pueda reconstruir la decisión sin volver a pedir información ya presentada.

Cuando el valor queda por debajo del límite fijado para el tramo, el responsable deja la observación registrada y continúa con la aplicación. No se abre un caso, no se retiene el movimiento y no se avisa a la contraparte. La diferencia se considera aceptable.

## Fuera del límite

Si el valor supera el límite del tramo, el movimiento queda pendiente y Tesorería solicita el comprobante rectificativo antes de aplicar el pago.

## Registro

Cada observación conserva el identificador del comprobante, el período informado, el importe acreditado y el límite aplicado para decidir.
