# Aprobaciones de tesorería — Boreal

## Regla específica

Para el cliente Boreal, el asistente puede aplicar automáticamente pagos parciales de hasta USD 5.000,00 cuando la referencia bancaria coincide exactamente con una única factura abierta.

## Excepciones

Se requiere aprobación humana si el importe supera el límite, si existen dos facturas candidatas, si hay una retención no documentada o si el pago contiene una diferencia mayor a USD 10,00 respecto del saldo esperado.

## Registro

Cada aplicación automática debe conservar el identificador del movimiento, la factura seleccionada, el score de coincidencia y la versión de esta política utilizada para decidir.
